bash
====


### How To Install ###


Just clone this repository to `~/.bash` directory

```
cd ~
git clone https://github.com/miracle/bash.git .bash
```

Add the following line to your `~/.bash_profile`

``` 
. ~/.bash/bashrc
```

OR

```
source ~/.bash/bashrc
```
